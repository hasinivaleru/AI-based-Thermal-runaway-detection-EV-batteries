# AI-Thermal-runaway
# NASA Battery Remaining Useful Life Prediction

This repository contains my implementation and experimental work on battery health analysis and Remaining Useful Life (RUL) prediction using the NASA Battery Dataset available on Kaggle.

The project focuses on applying machine learning and deep learning techniques to analyze battery degradation patterns and predict battery performance over charging and discharging cycles.

## Project Overview

The goal of this project is to study battery degradation behavior and develop predictive models for estimating battery life and performance.

The implementation includes:

* Data preprocessing and cleaning
* Exploratory Data Analysis (EDA)
* Battery degradation analysis
* Feature extraction
* Machine Learning / Deep Learning model training
* Remaining Useful Life (RUL) prediction experiments
* Model evaluation and performance analysis
